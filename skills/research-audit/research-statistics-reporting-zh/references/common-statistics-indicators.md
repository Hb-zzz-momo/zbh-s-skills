# 常用科研统计指标参考

## 快速选择

| 场景 | 推荐指标 | 说明 |
|---|---|---|
| 连续变量，分布近似对称 | Mean ± SD（均值±标准差） | 适合描述集中趋势和离散程度 |
| 连续变量，偏态或有异常值 | Median (IQR)（中位数和四分位距） | 比均值更稳健 |
| 分类变量 | n (%)（频数和百分比） | 表示类别计数和比例 |
| 比较差异大小 | Effect size + 95% CI（效应量和95%置信区间） | 说明差异多大、估计是否稳定 |
| 显著性检验 | P-value（P值） | 只能说明统计显著性，不能单独说明实际意义 |
| 模型/算法结果 | Mean ± SD、Median/IQR、rank、win/tie/loss | 视 run 数和分布而定 |

## 写法原则

- 描述性统计先看变量类型和分布。
- P 值要配合效应量和置信区间。
- 多算法多数据集对比要给逐数据集结果，不只给平均排名。
- 异常值多时优先报告中位数和 IQR。

## 论文句式

- 连续变量近似正态：`结果以均值±标准差表示。`
- 偏态变量：`结果以中位数（四分位距）表示。`
- 分类变量：`分类变量以频数（百分比）表示。`
- 差异报告：`组间差异同时报告效应量及95%置信区间。`

## 常见错误

- 所有变量都用均值±标准差。
- 只报告 P 值，不报告效应量。
- 把“不显著”写成“没有差异”。
- 算法实验只看均值，不看标准差、rank 和稳定性。
